什么是科学上网？

首先，我们需要先了解一下什么是科学上网？
Chatgpt对此的回答是“科学上网”通常指的是在中国大陆地区，通过一些技术手段绕过网络审查和限制，访问被屏蔽的国外网站和服务。由于中国实施的“防火长城”（Great Firewall）会限制对部分国际网站的访问（例如谷歌、Facebook、Twitter、YouTube等），因此许多人会通过使用虚拟专用网络（VPN）、代理服务器、或其他加密工具来实现“科学上网”。

因此通过正常的手段是无法浏览外网的，所以此时我们就需要使用一些加密手段的方式了————代理

Windows端代理

Windows端的代理软件我推荐的有两个，一是V2rayn，还有一就是clash系列的
接下来先介绍clash系列的。

但令人不幸的是，clash的作者于2024年3月份的时候突然在GitHub上删库了！因此clash的原内核已经停止更新维护了，时间一久可能就会出现一些漏洞与bug，同时也会不太稳定。但是clash的内核现在又被其他大佬进行修改并持续更新，属于是clash的分支，现在已经是非常完善了，也是我目前最推荐的代理app了。

Clash Verge

Clash Verge是目前最稳定的clash内核代理软件了，它的操作简便，上手很轻松，也是站长现在正在用的，其在GitHub上也有开源项目，并且保持着十分频繁的更新，所以会更加稳定。接下来会演示Clash Verge的下载使用。

Clash Verge的GitHub开源项目仓库：https://github.com/clash-verge-rev/clash-verge-rev

注：GitHub在国内的访问速度很慢，甚至有很大的概率会登不上去

进入到Clash Verge的GitHub仓库后，向下滑会看到Release的选项，点进去后会来到他的安装包下载界面

在里面你可以选择你电脑对应系统的安装包，并下载下来，Windows10及以上的选择这个下载就行了（现在应该没人用win7了吧）

下载下来后解压安装到桌面即可。

导入订阅使用

下载安装后，打开Clash Verge，会看到这样的界面

接着我们点击左边的订阅，将自己的订阅链接复制进去，然后点击导入，等到出现订阅成功的弹窗后，就表明导入成功了，如果失败，就检查一下自己的订阅链接是否正确或者到期。

接着点击左边栏的代理，如果出现了节点信息，就说明成功了，然后选择一个节点，双击选中

选中完后，再点击左边栏的设置，将系统代理打开，然后就配置成功可以使用了

其他设置

你也可以对Clash Verge进行其他一些设置。

  • 开机自启
    即开机后会自动启动Clash Verge，免去需要打开的繁琐步骤
  • 静默启动
    启动时不会弹出Clash Verge的界面
  • Tun模式
    相当于全局代理模式，代理程序会创建一张虚拟网卡，通过配置操作系统的路由将网络请求重定向到这张虚拟网卡，代理程序从虚拟网卡中读取并处理这些网络请求。
    使用Tun模式时需要先安装服务模式才能进行使用

以上是Clash Verge的主要设置，你还可以更改他的主题，进行个性化设置

常见问题

无法启动/不显示界面/闪退/只显示托盘图标

问题原因

  • Tauri 框架依赖于 WebView2。如果卸载或禁用了 WebView2 ,将无法显示界面。具体表现为: 程序可以启动，但是点击托盘菜单没有反应。
    解决方案

  • 如果是利用第三方软件禁用了 Edge，请检查是否同时禁用了 WebView2，将 WebView2 取消禁用。

  • 如果是卸载了 WebView2，可以下载 WebView2 安装包，重新安装 WebView2。

  • 如果是企业版系统或 Win7 无法安装 WebView2，请尝试在 Release 下载内置了 WebView2 的版本（带有 fixed_webview2 字样的安装包）。

  • 若问题仍然存在，请尝试使用 Windows 7 兼容模式启动。

Webview2 无法正常安装

问题原因

  • 可能是你的 Windows 系统关闭了自动更新。
    解决方案

  • 打开自动更新。

  • 若实在无法安装 WebView2，可以尝试在Clash Verge的 Release 下载内置了WebView2的版本（带有 fixed_webview2 字样的安装包）。

Android平台代理

安卓端代理APP目前我最推荐的便是Surfboard了，俗称冲浪板，他是迄今为止安卓平台上算较为稳定的代理软件了，出来的比较晚了，今年才出来的。但他的体系已经是完善的相对好的了。因此Surfboard是我首推的安卓端代理app了。

下载安装

接下来演示surfboard的下载安装教程

Surfboard官网的Github仓库开源项目地址: https://github.com/getsurfboard/surfboard

Surfboard官网（如果遇到相关问题可以去他们的官网寻求解决方案，也可以给我留言）：https://manual.getsurfboard.com/

进入到Surfboard的GitHub仓库后还是点击Release，进入后选择tv-release.apks下载安装

导入订阅使用

打开Surfboard后，点击下方栏的配置.

然后点击+号，选择从URL导入，接着将自己的订阅链接导入进去

返回主界面后，点击开始，接着会跳出是否创建VPN的选项，点击是，然后就可以使用了。
